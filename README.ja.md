# fukui_old_train

福井の昔ながらの旅客列車の運転体験を再現するFlutterアプリケーションです。このアプリは、デバイスのカメラと位置情報サービスを利用して、没入感のあるシミュレーションを提供します。

## 機能

- **拡張現実（AR）ビュー:** レトロな列車からの車窓をシミュレートし、実際のカメラ映像にアニメーションする煙突やランプを重ねて表示します。
- **列車アニメーション:** 列車の発車や停車のアニメーションを実装し、シミュレーションの臨場感を高めます。
- **位置情報データの活用:** デバイスのGPS位置情報を取得し、距離や方角の情報を表示します。

## 対応プラットフォーム

このプロジェクトは以下のプラットフォームで実行できるよう設定されています:
- iOS
- Android
- Web
- Windows
- macOS
- Linux

## 必須要件

- Dart SDK: `2.17.6` 以上
- Flutter SDK: `3.0.0` 以上

## はじめに

1.  **Flutter環境のセットアップ**
    - 手順については、[公式Flutterドキュメント](https://flutter.dev/docs/get-started/install)を参照してください。

2.  **リポジトリのクローン:**
    ```sh
    git clone https://github.com/your-username/fukui_old_train.git
    cd fukui_old_train
    ```

3.  **依存関係のインストール:**
    ```sh
    flutter pub get
    ```

4.  **アプリの実行:**
    ```sh
    flutter run
    ```
