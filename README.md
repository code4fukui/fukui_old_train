# fukui_old_train

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A Flutter application that recreates the experience of operating old-style passenger trains in Fukui, Japan. This app uses the device's camera and location services to create an immersive simulation.

## Features

- **Augmented Reality View:** Simulates the view from a vintage train, overlaying animated smokestacks and lamps onto the live camera feed.
- **Train Animations:** Includes animations for train departures and stops to enhance the simulation.
- **Location-Based Data:** Retrieves the device's GPS location to display distance and direction information.

## Supported Platforms

This project is configured to run on the following platforms:
- iOS
- Android
- Web
- Windows
- macOS
- Linux

## Requirements

- Dart SDK: `2.17.6` or higher
- Flutter SDK: `3.0.0` or higher

## Getting Started

1.  **Set up your Flutter environment.**
    - For guidance, see the [official Flutter documentation](https://flutter.dev/docs/get-started/install).

2.  **Clone the repository:**
    ```sh
    git clone https://github.com/code4fukui/fukui_old_train.git
    cd fukui_old_train
    ```

3.  **Install dependencies:**
    ```sh
    flutter pub get
    ```

4.  **Run the app:**
    ```sh
    flutter run
    ```
